# Authentication Standards

This folder contains authentication standards, implementation guides, and best practices for backend services.

## Purpose

Define organization-wide standards for:
- User authentication mechanisms
- Token-based authentication (JWT)
- Identity provider integration (Keycloak, OAuth2, SAML)
- Security best practices
