# Kubernetes Infrastructure Standards

This folder contains standards and guidelines for Kubernetes cluster setup, configuration, and management.

## Purpose

Define organization-wide standards for:
- Kubernetes cluster architecture
- Deployment configurations
- Networking and security
- Resource management
- Best practices

## Documents in This Folder

- `cluster-setup.md` — Cluster installation and configuration guide
- `deployment-standards.md` — Deployment manifest standards
- `networking.md` — Network policies and service mesh configuration

## Cluster Versions

Current supported versions:
- Production: Kubernetes 1.28+
- Development: Kubernetes 1.27+
